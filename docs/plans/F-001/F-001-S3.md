**Ghost movement algorithm:**
- Pathfinding is greedy: choose the adjacent open tile that minimises Manhattan
  distance to target, excluding the tile the ghost just came from (prevents
  oscillation). Ghost house tiles block normal ghosts (except when exiting).
- In Frightened mode: move to a random adjacent open tile each step, regardless
  of personality.
- In Scatter mode: target the ghost's fixed scatter corner.
- Ghost house exit thresholds (dots eaten):
  - Blinky: starts outside (no threshold)
  - Pinky: exits after 0 dots eaten
  - Inky: exits after 30 dots eaten
  - Clyde: exits after 60 dots eaten

## Acceptance criteria (subset)

- `Ghost::step(maze, player_pos, player_facing, blinky_pos, dots_eaten, tick)`:
  - On even ticks only (tick % 2 == 0), the ghost advances one tile
  - In Chase mode: selects target according to personality table
  - In Scatter mode: targets scatter corner
  - In Frightened mode: random adjacent open tile
  - Never reverses direction (excludes tile just came from), except when mode
    changes
  - Clyde switches between chase and scatter target based on Chebyshev distance
    to player
  - Ghosts inside the house (`is_ghost_house()` true) do not step until their
    dot threshold is met
- `GameTimers::tick() -> Vec<TimerEvent>`:
  - Decrements the active phase timer each tick
  - Emits `PhaseChanged` when scatter/chase phase expires
  - Emits `FrightenedBlinking` when frightened ticks drop to 30
  - Emits `FrightenedExpired` when frightened ticks reach 0
  - Emits `DeathPauseExpired` when death_pause_ticks reach 0
  - Emits `ExtraLifeDisplayExpired` when extra_life_display_ticks reach 0
- `GameTimers::start_frightened()` sets frightened_ticks = 100 and records the
  remaining phase ticks for resumption after frightened ends
- `GameTimers::end_frightened()` restores interrupted phase ticks
- `GameTimers::start_death_pause()` sets death_pause_ticks = 10
- `GameTimers::start_extra_life_display()` sets extra_life_display_ticks = 20

// src/game/ghost.rs
use crate::game::{Direction, GhostMode, Position};
use crate::game::maze::Maze;

#[derive(Debug, Clone, Copy, PartialEq)]
pub enum Personality { Blinky, Pinky, Inky, Clyde }

pub struct Ghost {
    pub pos: Position,
    pub mode: GhostMode,
    pub personality: Personality,
    pub home_pos: Position,       // ghost house start
    pub scatter_target: Position, // fixed corner
    pub prev_pos: Option<Position>,
    pub is_in_house: bool,
}

impl Ghost {
    pub fn new(personality: Personality, start: Position, scatter: Position) -> Self;
    /// Advance ghost one step (only acts on even ticks).
    pub fn step(
        &mut self,
        maze: &Maze,
        player_pos: Position,
        player_facing: Option<Direction>,
        blinky_pos: Position,
        dots_eaten: u32,
        tick: u64,
    );
    pub fn send_home(&mut self);
    pub fn set_mode(&mut self, mode: GhostMode);
    pub fn is_frightened_blinking(&self, frightened_ticks_remaining: u32) -> bool;
}
